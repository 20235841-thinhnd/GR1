**Tên đề tài:** Web dạy tiếng Việt VietStudy theo phương pháp ngữ âm học.<br>
**Mô tả:** Ứng dụng web VietStudy cung cấp môi trường học tập online, ứng dụng này cho phép trẻ em người Việt tại Việt Nam hoặc tại nước ngoài có thể học tiếng Việt theo phương pháp ngữ âm học.<br>
**Kiến thức áp dụng:**
- Kiến trúc: Microservices;
- Front-end: ReactJS;
- Back-end: NodeJS - CI/CD;
- Test: Unit Test.
